Sistema de Gestión del Conocimiento

Plataforma centralizada para la administración de la estructura académica institucional, procesos de acreditación y estrategias pedagógicas.

Módulo 1.2 Enfoque: Innovación Curricular Caso creado por: Carlos Castro

Descripción General

El sistema gestiona la estructura académica desde universidades y facultades hasta programas académicos. Soporta la administración de registros calificados, enfoques pedagógicos, pasantías, premios y alianzas institucionales con entidades externas (aliados), asegurando la calidad y normativa vigente.

Arquitectura:
Este módulo opera con una base de datos independiente y gestión de usuarios propia.

Ejes de Gestión

Estructura Académica

Gestión de Sedes, Facultades y Programas Académicos a nivel nacional.

Registros Calificados

Control de acreditación y cumplimiento de normativas institucionales.

Modelos Pedagógicos

Estrategias, características de innovación y enfoques curriculares.

Alianzas y Aliados

Administración de entidades externas y convenios institucionales.

Pasantías y Premios

Seguimiento a actividades externas, premios y reconocimientos.

Acceso y Seguridad

Control de usuarios y permisos específicos para el módulo.

Sobre el Desarrollo

La plataforma integra lo mejor de dos mundos: un núcleo de procesamiento de datos desarrollado en FastAPI (utilizando patrones de Modelos y Repositorios) y una interfaz web robusta creada con Flask. Esta integración permite que el sistema de Innovación Curricular sea escalable, eficiente y fácil de administrar.

Nota de Implementación: Para optimizar la experiencia de usuario, el módulo de Gestión de Usuarios utiliza una arquitectura de Single Page CRUD, mientras que el resto de los módulos emplean un sistema de navegación tradicional de dos páginas por tabla para garantizar claridad y orden en los procesos académicos.
